\subsection{Detailed Description}
Base class for error code handling. 

This class stores the last error code that happened to a derived class. It two public methods to check the existence and value of the last error code and two protected methods to set and clear the code. This base class also defines N\+O\+\_\+\+E\+R\+R\+O\+R\+\_\+\+C\+O\+D\+E (value = N\+U\+L\+L) to signal a no error condition.

\begin{DoxyNote}{Note}
For performance reasons, the error code is stored size\+\_\+t. 
\end{DoxyNote}

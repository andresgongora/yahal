Detailed Description

template<typename T_NODE>
class yahal::utility::oop::LinkedListNode< T_NODE >

Base class for classes that can be linked in a LinkedList.

The Curiously Recurring Template Pattern is used to acces the derived class T_NODE. All methods for LinkedListNode are private and only accesible by LinkedList.
